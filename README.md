# 🇪🇺 awesome-europe - Open source tools for Europe

## 🌍 What this is

awesome-europe is a curated list of open source software for Europe. It brings together tools and projects for public services, cross-border work, digital rules, data sharing, and common standards.

Use it to find software that fits European needs, such as:

- open data tools
- eHealth systems
- GDPR-friendly software
- eIDAS and digital identity tools
- standard-based platforms
- cross-border service tools
- public sector software
